# Fibonacci Calculator Skill
## Overview
The Fibonacci Calculator skill is designed to calculate Fibonacci numbers. This skill takes an integer as input and returns the corresponding Fibonacci number.

## Usage
To use the Fibonacci Calculator skill, simply run the `main.py` file and enter the position of the Fibonacci number you want to calculate when prompted.

## Example Use Cases
* Calculate the 10th Fibonacci number
* Determine the 20th Fibonacci number
* Find the 5th Fibonacci number

## Code Structure
The code for this skill is structured into a single Python file, `main.py`, which contains the `fibonacci` function and the `main` function. The `fibonacci` function calculates the nth Fibonacci number, and the `main` function handles user input and output.

## Requirements
The Fibonacci Calculator skill requires Python 3.x to run. The required libraries are listed in the `requirements.txt` file.
